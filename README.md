Simple Hospital Web Page (HTML Practice)
This project is a basic static website built entirely with HTML, without using any frameworks or external libraries. It was created as a practice exercise to strengthen fundamental web development skills and demonstrate how to structure a simple webpage.

The page represents a hospital-themed website with basic sections such as:

Home

Services

Doctors

Contact information

It focuses on clean structure and semantic HTML, serving as a foundation for future projects that may include CSS styling or JavaScript functionality.

Pure HTML: No frameworks, libraries, or external dependencies.

Basic navigation: Simple links between sections.

Structured content: Organized with headings, paragraphs, and lists.

Practice-oriented: Designed to reinforce core HTML concepts.

The goal of this project is to:

Practice writing semantic and well-structured HTML.

Understand how to build a static webpage without relying on frameworks.

Provide a starting point for expanding into CSS and JavaScript
